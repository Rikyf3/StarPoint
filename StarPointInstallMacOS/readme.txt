Install procedure for MacOS

1. Copy simple_frozen_graph.pb in Applications/PixInsight/library/
2. Copy StarPoint-pxm.dylib in Applications/PixInsight/bin/
3. Open PixInsight
4. Go to Process -> Modules -> Install Modules...
5. Click Search
6. Click Install
7. StarPoint module can now be found under Process -> Render -> StarPoint
